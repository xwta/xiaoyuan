# 运营能力升级完成说明

## 1. 本轮目标

本轮继续围绕系统运营能力补齐，重点处理：

- 运营只读接口
- 代理点商品数量读取
- 校区楼栋读取
- 流程记录读取
- 迁移执行器
- 后台运营组件

## 2. 新增后端接口文件

```text
server/src/routes/areaRead.js
server/src/routes/agentGoodsRead.js
server/src/routes/recordRead.js
server/src/routes/orderReadV2.js
```

## 3. 已挂载接口

以下接口已挂载到后端入口：

```text
GET /api/ops/areas
GET /api/ops/blocks
GET /api/ops/goods
GET /api/ops/records/:bizId
```

用途：

- 校区读取
- 楼栋读取
- 代理点商品数量读取
- 业务流程记录读取

## 4. 已新增后台 API 方法

```text
fetchOpsAreas
fetchOpsBlocks
fetchOpsGoods
fetchOpsRecords
```

文件：

```text
admin/src/api.js
```

## 5. 已新增后台组件

```text
admin/src/components/OpsManager.vue
```

组件包含：

- 校区列表
- 宿舍楼列表
