# 数据库迁移说明

## 1. 脚本目录

基础脚本：

```text
database/mysql_init.sql
```

升级脚本：

```text
database/migrations
```

当前包含：

```text
001_flow_record.sql
002_agent_count.sql
003_agent_scope.sql
```

## 2. 执行步骤

进入后端目录：

```bash
cd server
```

安装依赖：

```bash
npm install
```

确认已配置 `.env`：

```text
DB_ENABLED=true
```

执行：

```bash
npm run migrate
```

## 3. 当前新增结构

```text
flow_record
agent_product_count
agent_building_scope
```

## 4. 使用建议

- 按文件名顺序执行。
- 新增结构时继续在 migrations 目录下增加新文件。
- 不建议修改已经使用过的脚本。
