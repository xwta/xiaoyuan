# 前后端联调说明

## 1. 后端启动

进入后端目录：

```bash
cd server
npm install
npm run dev
```

默认服务地址：

```text
http://localhost:3000
```

健康检查：

```text
GET http://localhost:3000/api/health
```

## 2. 小程序接口地址

小程序接口地址在：

```text
miniprogram/app.js
```

当前配置：

```js
apiBaseUrl: 'https://example.com/api'
```

本地联调时，需要改成：

```js
apiBaseUrl: 'http://localhost:3000/api'
```

如果使用微信开发者工具，需要在详情设置里临时勾选：

```text
不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

## 3. 已接入接口的小程序页面

### 学生端

| 页面 | 文件 | 接口 |
|---|---|---|
| 首页 | `pages/home/index.js` | `GET /api/products` |
| 分类页 | `pages/category/index.js` | `GET /api/categories`, `GET /api/products` |
| 购物车 | `pages/cart/index.js` | 本地 Storage |
| 确认订单 | `pages/order/confirm.js` | `POST /api/orders` |
| 订单列表 | `pages/order/list.js` | `GET /api/orders` |
| 订单详情 | `pages/order/detail.js` | `GET /api/orders/:id` |

### 代理端

| 页面 | 文件 | 接口 |
|---|---|---|
| 代理首页 | `pages/agent/index.js` | `GET /api/agent/summary` |
| 代理订单 | `pages/agent/orders.js` | `GET /api/agent/orders`, `POST /api/agent/orders/:id/accept`, `POST /api/agent/orders/:id/delivered` |
| 库存管理 | `pages/agent/stock.js` | `GET /api/agent/stocks` |
| 佣金中心 | `pages/agent/commission.js` | `GET /api/agent/commissions` |

## 4. 购物车逻辑

购物车目前使用微信小程序本地缓存：

```text
campus_snack_cart
```

相关工具文件：

```text
miniprogram/utils/cart.js
```

已支持：

- 获取购物车
- 保存购物车
- 添加商品
- 修改数量
- 清空购物车
- 计算合计金额

## 5. 当前订单流程

```text
首页/分类页加购
→ 购物车修改数量
→ 确认订单
→ POST /api/orders 创建订单
→ 清空购物车
→ 跳转订单详情
→ 订单列表可查看订单
→ 代理端可接单/完成订单
```
