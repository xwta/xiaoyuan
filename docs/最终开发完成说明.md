## 6. 当前完整链路

当前项目已支持：

```text
用户登录
→ 商品浏览
→ 加入购物车
→ 新增寝室地址
→ 提交订单
→ 查看订单
→ 代理点接单
→ 完成订单
→ 后台查看数据
→ 后台管理商品 / 订单 / 代理点
```
