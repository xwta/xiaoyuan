# 运行说明

## 后端服务

```bash
cd server
npm install
npm run dev
```

访问：

```text
http://localhost:3000
```

健康检查：

```text
GET /api/health
```

后端当前已拆分：

```text
server/src/routes/products.js
server/src/routes/campus.js
server/src/routes/orders.js
server/src/routes/agent.js
server/src/mock/store.js
```

## 小程序端

使用微信开发者工具打开：

```text
miniprogram
```

接口环境配置：

```text
miniprogram/config/env.js
```

默认本地接口：

```text
http://localhost:3000/api
```

## 管理后台

```bash
cd admin
npm install
npm run dev
```

访问：

```text
http://localhost:5173
```

后台已包含：

- Vue + Vite
- Element Plus
- 数据看板
- 商品列表
- 最近订单
- 代理统计

## 数据库

数据库脚本：

```text
database/schema.sql
```

当前后端仍使用演示数据，下一步接入 MySQL。

## 推荐启动顺序

```text
1. 启动 server
2. 启动 admin
3. 打开 miniprogram
```
