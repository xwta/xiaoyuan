# 运行说明

## 1. 小程序端

使用微信开发者工具打开 `miniprogram` 目录。

当前小程序端是页面骨架版本，已包含：

- 首页
- 购物车
- 订单确认
- 代理点订单

后续需要接入真实接口地址：

```js
App.globalData.apiBaseUrl = '你的后端接口地址'
```

## 2. 后端服务

进入后端目录：

```bash
cd server
npm install
npm run dev
```

默认启动地址：

```text
http://localhost:3000
```

健康检查：

```text
GET http://localhost:3000/api/health
```

## 3. 数据库

数据库脚本在：

```text
database/schema.sql
```

建议使用 MySQL 8.x。
