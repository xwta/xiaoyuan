# 校园零食售卖平台小程序产品方案

## 1. 项目定位

校园零食售卖平台是一个面向高校学生的即时零售小程序，支持学生在线购买零食、饮料、泡面、宿舍日用品等商品，并通过校园代理点、宿舍楼代理或校园小仓完成配送到寝室或代理点自提。

核心目标：

- 学生端：不用出宿舍即可购买零食和日用品。
- 代理端：校园代理点可以接单、配送、自提核销并赚取佣金。
- 平台端：统一管理商品、库存、订单、代理点、佣金和多校区运营。

## 2. 目标用户

### 2.1 学生用户

典型需求：

- 夜宵、饮料、泡面、零食即时购买。
- 不想下楼，希望送到寝室。
- 希望价格透明、配送快、能自提省配送费。
- 考试周、宿舍聚会、日用品临时缺货等场景下快速下单。

### 2.2 校园代理点

代理点可以是宿舍楼学生代理、校园创业团队、校内便利店、校园小仓或跑腿团队。

典型需求：

- 获取稳定订单来源。
- 接单配送赚佣金。
- 操作简单，结算清晰。
- 可查看订单、库存和收益。

### 2.3 平台运营方

典型需求：

- 统一管理商品和库存。
- 管理校区、宿舍楼、代理点。
- 管理订单履约和售后。
- 管理佣金结算和运营数据。
- 支持多学校、多校区复制。

## 3. 商业模式

### 3.1 平台自营库存 + 代理点配送

平台统一采购商品，放在校内小仓或代理点，由代理点负责配送和自提核销。

优点：商品、价格和服务可控，平台利润空间高。
缺点：需要库存和补货管理。

### 3.2 代理点自备库存 + 平台抽佣

代理点自己维护库存和履约，平台提供系统、订单和结算能力。

优点：平台轻资产，扩张快。
缺点：商品质量、价格和库存更难统一。

### 3.3 推荐模式：混合模式

平台统一管理核心爆品，代理点可维护本楼库存。

推荐商品：

- 饮料水饮
- 泡面速食
- 薯片膨化
- 饼干糕点
- 糖果巧克力
- 宿舍日用品
- 夜宵套餐

## 5. 核心业务流程

### 5.1 学生下单流程

```text
进入小程序
→ 选择学校/校区
→ 浏览商品
→ 加入购物车
→ 选择配送方式：送到寝室 / 代理点自提
→ 填写校区、宿舍楼、寝室号
→ 系统匹配代理点
→ 提交订单
→ 微信支付
→ 代理点接单
→ 配送或自提核销
→ 订单完成
```

### 5.2 送到寝室流程

```text
学生选择送到寝室
→ 填写校区、宿舍楼、寝室号、手机号
→ 系统匹配服务该宿舍楼的代理点
→ 支付成功
→ 代理点收到订单
→ 代理点接单
→ 代理点备货
→ 配送到寝室
→ 点击确认送达
→ 用户确认或超时自动完成
```

订单状态：

```text
待支付 → 已支付待接单 → 已接单 → 配送中 → 已送达 → 已完成
```

### 5.3 代理点自提流程

```text
学生选择代理点自提
→ 选择附近代理点
→ 支付成功
→ 系统生成核销码
→ 学生到代理点取货
→ 代理点扫码或输入核销码
→ 订单完成
```

订单状态：

```text
待支付 → 已支付待备货 → 待自提 → 已核销 → 已完成
```

## 9. 订单分配逻辑

优先级：

1. 用户宿舍楼绑定的默认代理点。
2. 该宿舍楼可服务的代理点。
3. 当前库存充足的代理点。
4. 距离最近的代理点。
5. 当前接单状态正常的代理点。

异常处理：

- 无代理点：提示当前楼栋暂不支持配送。
- 库存不足：推荐其他代理点或提示部分商品不可购买。
- 代理点超时未接单：自动转派其他代理点。
- 无可转派代理点：用户可取消退款或平台人工处理。

## 13. 首批商品建议

饮料类：可乐、雪碧、冰红茶、绿茶、矿泉水、功能饮料、酸奶。

速食类：桶装泡面、火鸡面、螺蛳粉、自热米饭、面包、火腿肠。

零食类：薯片、辣条、饼干、巧克力、糖果、坚果、蛋糕、肉脯。

日用品：纸巾、湿巾、牙刷、牙膏、洗衣液小包装、卫生巾、一次性筷子、垃圾袋。

## 14. 当前落地重点

第一版不要追求大而全，优先打通三件事：

1. 学生能顺畅下单。
2. 代理点能顺畅接单。
3. 平台能看清订单、库存和佣金。

先跑通交易闭环，再逐步增加优惠券、秒杀、代理招募、佣金提现和多校区复制。
