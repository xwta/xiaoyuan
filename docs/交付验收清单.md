# 交付验收清单

## 一、当前可验收内容

### 1. 小程序学生端

- 首页可展示商品
- 分类页可展示分类商品
- 商品可加入购物车
- 购物车可调整数量
- 确认订单页可计算商品金额、配送费、实付金额
- 可提交订单
- 可查看订单列表
- 可查看订单详情
- 可进入寝室地址页面
- 可进入个人中心页面

### 2. 小程序代理端

- 可查看代理首页统计
- 可查看代理订单列表
- 可查看代理订单详情
- 可进行接单操作
- 可进行订单完成操作
- 可进行自提取货处理
- 可查看库存列表
- 可查看佣金中心

### 3. 后端服务

- Express 服务可启动
- 健康检查接口可用
- 商品接口可用
- 订单接口可用
- 校区接口可用
- 宿舍楼接口可用
- 代理点接口可用
- 代理端接口可用
- 后台接口可用
- 支持 mock 数据模式
- 支持 MySQL 数据模式

### 4. 管理后台

- 后台可启动
- 数据看板可查看
- 商品列表可查看
- 商品可新增
- 商品可编辑
- 商品可上下架
- 商品可移除
- 订单列表可查看
- 订单状态可修改
- 代理点列表可查看
- 代理点可新增
- 代理点可启用 / 停用

### 5. 数据库

- 提供 MySQL 初始化脚本
- 提供数据库说明文档
- 提供环境变量示例
- 商品模块支持数据库切换
- 订单模块支持数据库切换
- 校区、楼栋、代理点模块支持数据库切换

### 6. 部署与运行

- 后端可本地启动
- 后台可本地启动
- 小程序可在微信开发者工具打开
- MySQL + 后端支持 Docker Compose 启动

## 二、上线前必须补齐

### 1. 用户体系

- 微信登录
- 用户唯一身份绑定
- 用户手机号授权或录入
- 用户默认寝室地址

### 2. 支付体系

- 微信支付
- 支付回调
- 支付状态校验
- 订单超时取消

### 3. 后台安全

- 后台登录
- 管理员权限
- 接口鉴权
- 操作日志

### 4. 订单能力

- 订单搜索
- 订单分页
- 订单退款
- 售后处理
- 配送记录

### 5. 代理体系

- 代理点账号
- 代理点权限隔离
- 代理佣金规则
- 佣金结算
- 提现记录

### 6. 商品和库存

- 商品图片上传
- 商品上下架审核
- 库存扣减
- 低库存预警
- 多代理点库存

### 7. 运营能力

- 优惠券
- 满减活动
- 新人首单优惠
- 秒杀商品
- 数据图表

## 三、当前版本结论

当前版本可以作为：

- 校园零食平台 MVP 骨架
- 小程序 + 后端 + 后台一体化二开底座
- 开发排期和报价演示版本
- 后续接入真实业务的基础版本

当前版本不建议直接作为正式商用版本上线。

正式上线前，至少需要补齐：

```text
微信登录
微信支付
后台登录鉴权
真实数据库完整接入
订单异常处理
生产环境部署配置
```
