# 校园零食售卖平台小程序

基于开源点餐/外卖平台思路改造的校园零食售卖小程序项目。

## 项目定位

面向校园场景，支持学生在线购买零食、饮料、泡面、夜宵等商品，支持普通电商下单、校园代理点下单、配送到寝室、自提核销、代理点订单管理与佣金统计。

## 核心功能

### 学生端

- 商品浏览
- 分类筛选
- 购物车
- 在线下单
- 寝室地址管理
- 配送到寝室
- 代理点自提
- 订单状态查询

### 校园代理点端

- 代理点登录
- 查看分配订单
- 接单/配送/完成
- 自提扫码核销
- 销售数据统计
- 佣金统计

### 后台管理端

- 商品管理
- 分类管理
- 库存管理
- 校区管理
- 宿舍楼管理
- 代理点管理
- 订单管理
- 配送范围管理
- 佣金规则配置

## 目录结构

```text
miniprogram/   微信小程序端
server/        后端服务
admin/         管理后台
database/      数据库脚本
docs/          项目说明与二开文档
```

## 推荐二开来源

推荐参考开源项目：`woniudiancang/bee`。

该项目原本偏点餐/外卖场景，适合改造成校园零食平台：

- 商品下单逻辑可复用
- 配送/自提逻辑可复用
- 门店概念可改成校园代理点
- 核销能力可改成代理点自提核销

## 当前版本说明

当前仓库已初始化为校园零食平台二开启动版，后续可以继续接入完整小程序页面、后台管理和服务端接口。
